# SkillBridge

SkillBridge is a platform designed to connect volunteers with NGOs, enabling meaningful collaboration and social impact.

## Features
- Volunteer registration and login
- NGO onboarding
- Opportunity discovery
- Real-time communication

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Setup
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
