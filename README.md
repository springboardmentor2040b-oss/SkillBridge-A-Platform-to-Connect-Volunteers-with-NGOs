# SkillBridge

## About
A web platform connecting skilled volunteers with NGOs for meaningful opportunities.

## Features
- User registration (Volunteers & NGOs)
- Opportunity posting and browsing
- Application management system
- Responsive design for all devices

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB

## Setup
```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
